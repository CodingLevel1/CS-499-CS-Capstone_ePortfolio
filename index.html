CS-410 Software Reverse Engineering

Category: Databases.

This project focuses on the implementation and optimization of key data structures and algorithms, particularly the Binary Search Tree (BST) and AVL Tree, to manage a collection of course data for a course planner system. The goal of the project was to efficiently store, retrieve, and manage course data while ensuring optimal performance in search and insertion operations, even with large datasets. The initial version of the project used a standard Binary Search Tree (BST) but faced limitations in performance as the dataset grew. To address these limitations, I enhanced the program by implementing an AVL Tree, a self-balancing binary search tree, to improve the efficiency of insertion and search operations. This enhancement reduced the time complexity of both operations from O(n) in the basic BST to O(log n) in the AVL Tree, which ensures better scalability when handling large amounts of course data. This project highlights my ability to apply algorithmic principles and data structure optimization in real-world scenarios. It also showcases my skills in addressing performance issues in software applications and improving the overall user experience through better scalability.
